Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

Selector Specificity

Selector specificity is a measure of how specific a CSS selector is in targeting HTML elements. It determines which CSS rules are applied to an element when multiple rules conflict. Specificity is calculated based on the components of a selector: the number of ID selectors, class selectors, and type selectors, with higher numbers indicating higher specificity.

Calculation: Specificity is typically represented as a four-digit number, where:

  • The first digit represents the number of ID selectors.
  • The second digit represents the number of class selectors, attribute selectors, and pseudo-classes.
  • The third digit represents the number of type selectors and pseudo-elements.
  • The fourth digit represents the number of inline styles.

Complex Selectors

Complex selectors are CSS selectors that combine multiple simple selectors to target elements more precisely. They allow you to define styles for elements based on their relationship with other elements or their attributes.

Examples of complex selectors include descendant selectors, child selectors, adjacent sibling selectors, and attribute selectors combined with other selectors.

Difference between Pseudo-classes and Pseudo-elements

Pseudo-classes: Pseudo-classes are keywords added to selectors that specify a special state of the selected element. They are used to style elements based on user interaction or document structure that can't be expressed using simple selectors alone. Examples include :hover, :active, :focus, :nth-child(), etc.

Pseudo-elements: Pseudo-elements are keywords added to selectors that create virtual elements in the document tree. They allow you to style parts of an element's content, such as the first line of a paragraph or the first letter of a word. Examples include ::before, ::after, ::first-line, ::first-letter, etc.
